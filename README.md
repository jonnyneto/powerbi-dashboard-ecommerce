# Dashboard de Vendas – Power BI

Projeto de Business Intelligence desenvolvido com Power BI utilizando dataset real de e-commerce.

## Objetivo
Analisar o desempenho de vendas e apoiar a tomada de decisão por meio de indicadores estratégicos.

## Principais Análises
- Faturamento total
- Lucro e margem de lucro
- Vendas ao longo do tempo
- Vendas por região
- Desempenho por categoria e produto
- Participação por canal de vendas

## Ferramentas Utilizadas
- Power BI
- Power Query
- DAX

## Destaques Técnicos
- Tratamento de dados no Power Query
- Criação de medidas em DAX
- Segmentadores interativos
- Dashboard orientado a negócio
